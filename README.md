# Renewal KC Web Generator

Aplikasi web modern berbasis dashboard untuk membantu pembuatan, penyesuaian, dan pencetakan dokumen **Renewal KC (Perpanjangan Kontrak Kerja / Key Customer)** secara otomatis, dinamis, dan terstruktur.

## Fitur Utama

- **Live Dynamic Preview**: Surat perpanjangan kontrak diperbarui secara langsung saat pengguna mengisi form input.
- **Modern Dashboard Design**: Antarmuka responsif dengan palet warna premium, efek glassmorphism, dan micro-animations.
- **Export to PDF / Direct Print**: Dilengkapi dengan styling CSS print (`@media print`) khusus untuk menghasilkan cetakan dokumen yang presisi dan bersih (tanpa layout dashboard).
- **History Logger**: Menyimpan riwayat dokumen yang telah dibuat ke dalam `LocalStorage` sehingga dapat dilihat kembali atau dihapus kapan saja.

## Teknologi

- **HTML5**: Struktur semantik.
- **Vanilla CSS3**: Custom properties (CSS variables), CSS grid/flexbox, efek glassmorphism, optimasi layout cetak.
- **Vanilla JavaScript**: Validasi form, sinkronisasi DOM, integrasi `LocalStorage`, interaksi modal & pencetakan.

## Cara Menjalankan

Cukup buka file `index.html` di browser Anda (Google Chrome, Microsoft Edge, Firefox, dll.). Tidak diperlukan server atau kompilasi khusus.
